# Discord Music Bot

A Python Discord music bot with YouTube and Spotify support.

## Features
- Play music from YouTube
- Queue system
- Pause, Resume, Skip
- Bass boost
- Slash commands
- Works on Mac & Render 24/7

## Installation
1. Clone this repository
2. Create a `.env` file and add your `DISCORD_TOKEN`
3. Install requirements:
   ```bash
   pip install -r requirements.txt
